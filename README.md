
```
Создать сотрудника → email = "existing@example.com" (уже есть) → Нажать SAVE
```

**Ожидаемая ошибка:**
```
⚠️ Сотрудник с таким email уже существует
```

**5. Дата в будущем:**

```
Создать сотрудника → hire_date = 2025-12-31 → Нажать SAVE
```

**Ожидаемая ошибка:**
```
⚠️ Дата приёма не может быть в будущем

```


**Ожидаемый результат:**
```
Страница 404 Not Found
```

**Что говорить:**
> "Я пытаюсь открыть сотрудника с ID 9999, которого не существует. Django автоматически показывает страницу 404 - это стандартная обработка ошибки."

**Показать код:**

```python
# views.py
def employee_update(request, pk):
    employee = get_object_or_404(Employee, pk=pk)  # ← Эта строка создаёт 404
    # ...
```

**Что говорить:**
> "В коде я использую функцию get_object_or_404. Если сотрудник не найден, она автоматически возвращает страницу 404."

### **КРИТЕРИЙ 8: Статика при DEBUG=False (10 баллов)**

#### **Что показывать комиссии:**

**Шаг 1:** Показать настройки статики

```python
# settings.py
MIDDLEWARE = [
    'whitenoise.middleware.WhiteNoiseMiddleware',  # ← Это важно!
    # ...
]

STATIC_URL = '/static/'
STATIC_ROOT = os.path.join(BASE_DIR, 'staticfiles')
STATICFILES_STORAGE = 'whitenoise.storage.CompressedManifestStaticFilesStorage'
```

**Что говорить:**
> "Мы используем whitenoise для отдачи статических файлов. Это middleware, который обрабатывает CSS, JS, изображения."

**Шаг 2:** Показать сборку статики

```bash

```

**Ожидаемый вывод:**
```
Copying '/path/to/static/admin/css/base.css'
Copying '/path/to/static/admin/css/dashboard.css'
...
136 static files copied.
```

**Что говорить:**
> "Команда collectstatic собирает все статические файлы в папку staticfiles."

**Шаг 3:** Установить DEBUG=False

```bash
# В .env
DEBUG=False
```

**Шаг 4:** Перезапустить сервер и открыть админку

```
http://127.0.0.1:8000/admin/
```

**Что говорить:**
> "Несмотря на DEBUG=False, админ-панель отображается со стилями. Это работает благодаря whitenoise."

**Шаг 5:** Показать, что без collectstatic стилей нет


**Ожидаемый результат:** Админка без стилей (голый HTML)

**Что говорить:**
> "Если не выполнить collectstatic, админка теряет стили. Это демонстрирует важность этого шага."
